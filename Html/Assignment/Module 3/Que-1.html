Q-1. Are the HTML tags and elements the same thing ? With Example .


Ans -> No Tag and Elements are not the same thing.


           Tag           : Tag used to show how data will show up on the Web page. Tag is contained in strat bracket and end bracket.

           Elements : Elements is the combination of <Start Tag> Content <End Tag>.

                         Eg.      Tag           :- <h1> </h1>

                                     Elements :- <h1>Mihir Kalariya</h1> This is Element
