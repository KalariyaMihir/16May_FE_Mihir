Q-5. What are different types of lists in HTML?


Ans -> There Are Three Types of List tag in HTMl.

- Unordered List :-

-> Unordered List used when you don't want the data in a manner or particular order.

Example :

  • Html
  • Css
  • Javascript
  • Bootstrap

- Ordered List :-

Ordered list used when we want the data in a particular Order like in a numeric manner.

Example :

  1. one
  2. Two
  3. Three
  4. Four
  5. Five

- Data List / Description List :-

Data list used when you want to add the additional information / definition to the data.
