Q-3. What are void elements in HTML? With Example


Ans : Before Void Elements we need to understand that There is Two types of tag.
           1.) Pair Tag
           2.) Non Pair Tag also called as( Void Elements / Self Tag)


           Void Elements :- Void element is self or non pair tag it does not comes with start and end tag it stands alone.

Eg. <br> , <b> , <meta> , <link> , etc...
