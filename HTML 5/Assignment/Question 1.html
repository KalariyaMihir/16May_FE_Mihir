Q.1 -> What are the new tags added in HTML5?

Ans ->

There are Two Types of Tag in HTML5 -> 1. Structure Type 2.Content

Structure Type tag's :

1.) Header

2.) Nav

3.) Section

4.) Summary

5.) Hgroup

6.) Aside

7.) Footer

Content Type tag's :

Address Detail Date Color Range Progress
Video Audio Poster Blockquote Del Sup
Sub Abbr kbd Ltr Rtl Ins
Pre Cite Datalist Bdo Blink Br
Svg Canvas Caption Button DD Dir
Legend Fieldset BGsound Base Meter Object
