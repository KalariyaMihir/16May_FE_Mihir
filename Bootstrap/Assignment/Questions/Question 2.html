- Containers are the most basic layout element in Bootstrap.
                        - Container required using default grid system.
                        - Types of Container Class 1.) container
                                                   2.) container-fluid
                                                   3.) container-{breakpoints} 

                        -> container - doesn't comes with 100% display width, it gives some blank space from left and right.
                        -> Container-fluid - it comes with the full 100% width.
                        -> container-{breakpoint} - it has 100% display width until the specified breakpoints.
