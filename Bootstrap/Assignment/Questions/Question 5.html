What is the difference between Bootstrap 4 and Bootstrap 5

Bootstrap 4 Bootstrap 5
- Use jQuery For some Functionality. - Removed jquery from Bootstrap 5.
- Limited classes for spacing and alignment. - Add more Utility classes.
- Limited Customization Option. - Enhanced Customization Option.
- Used Glyphicons. - Added SVG icons.
- Use Flexbox for Layout. - USe Grid for Layout.
- Use jQuery for functionality. - Use Modern Vanilla Javascript.
- less option for designing - Improve Typography for new design
