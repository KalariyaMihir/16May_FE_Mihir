Ans : - Pop up boxes are used to show the important information, take input or asking for confirmation in small window.
               - There are Three types of Pop-Up Boxes :- 1.) Alert 
                                                          2.) Prompt
                                                          3.) Confirm   

                Example : 1.) Alert : - Used to pass a simple alert message

                                    alert("Welcome to Our Site"); 
    
                          2.) Prompt : - Used to take input from users.
                                       - by default prompt's value is string datatype. 
                          
                                    prompt("Enter Your Name");

                          3.) confirm : - Used to take a confirmation from users.
                                        - Yes or Cancel.
                                        
                                    confirm("Are You above 18");
