Ans : - Loops are used to execute a block of code multiple times.
               - There are 6 types of Loops JS : 1.) For Loop
                                                 2.) While Loop
                                                 3.) Do-while Loop 
                                                 4.) For-in Loop
                                                 5.) For-of Loop
                                                 6.) For-each Loop
                
                note : printing 1 to 10 numbers for show the structure.
                                            
                1.) For Loop : structure of for loop ->
                                    
                                    for(let i = 1; i <= 10; i++)
                                        {
                                            console.log(i);
                                        }
                
                2.) While Loop : structure of while loop -> 
                                    
                                    let i = 1;
                                    while(i <= 10)
                                    {
                                        console.log(i);
                                        i++
                                    }
                                    
                3.) Do-While Loop : Structure of do-while Loop ->
                                    
                                    let i = 1;
                                    
                                    do{
                                        console.log(i);
                                        i++;
                                    }
                                    while(i <= 10);

                4.) For-In Loop : Structure of For-in Loop ->

                                    let car = ["Alto", "Zen", "i10", "Swift"]

                                    for(let i in car)
                                    {
                                        console.log("Budget Friendly car "+car[i]+"<br>");
                                    };
                                    
                5.) For-of Loop : Structure of For-of Loop ->

                                    let laptop = ['HP', 'DELL', 'Lenovo', 'ASUS'];
                                    for(let i of laptop)
                                    {
                                        console.log(i + ",")
                                    }
