Q-2. What are the disadvantages of css?

Ans - 1.) Browser compatibility - Different Browser has minor different rules , so we need to use Browser web kit to make css Browser compatible.

- 2.) Loading - Larger external css file takes to much time to load, it makes website slow.

- 3.) Edit - Make changes after applying css is a hard task to do.

- 4.) Animation - To create some kind of logical animation we need to use other languages like Js.
