Q-10. Why should background and color be used as separate properties?

Ans - 1.) Background -> Background property represent the elements background,
                   if we apply the background property and give value it applies in the elements background. Ex. h4{ background: value here; }

- 2.) Color -> Color property used to apply the color to the fonts. Ex. h4{ color: White; }

- for that reason both background and color property used separately.
