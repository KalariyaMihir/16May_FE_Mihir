Q-3. What is the difference between CSS2 and CSS3?

CSS2 CSS3
CSS2 is the upgraded version of css1, with better features then css1. CSS3 is the upgraded version of css2, with more better features then css2.
CSS2 has selectors like element, class, id, child etc. CSS3 comes with more advanced selectors like attribute, nth-child..
No media queries in css2 so we can not make website responsive using css2. In CSS3 there is concept of media queries we can create responsive website using media queries.
No Animation in css2. There is Animation Property in css3. we can create more attractive website using animation.
No concept of Grid and Flex. Concept of Flex and Grid by using it's easier to create website responsive
