Q-2. What are the ways to integrate CSS as a web page?

Ans - Basically there are three ways to integrate css into web pages.

1.) Inline - Inline CSS applied in the elements start tag with the "Style" attribute .

2.) Internal - Internal CSS applied in the head section, with the "Style" tag.

3.) External - External CSS applied in the head section, with the "link" tag's "href" attribute.
