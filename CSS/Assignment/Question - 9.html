Q-9. Which property controls the image scroll in the background?

Ans - Background Attachment property controls the scroll of image, it has value like "Scroll, Local, Fixed".

- Scroll,Local value is the default value when you scroll the page image will scroll up down to.

- Fixed value fixed the image in the position we set.
