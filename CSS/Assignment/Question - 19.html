Q-19. what is the rule set?

Ans - CSS rule set is the rules to apply the css to the html web page.

- it contains the selector , curly braces, property and its value.

- 1.) Selector -> Selector used to select HTML elements or specific elements where css will be applied.

- 2.) Curly Braces -> {}(curly braces) contains the property and it's value.

- 3.) Property -> Property defines which style will be applied. like "color".

- 4.) Value -> Specify the amount of the property . like "Coral".
