Q-17. What is the meaning of the CSS selector?

Ans - Selectors used for applied style to the html web page.

- with use of Selectors we can apply style to the particular elements, class, id...

- there are many types of Selectors :

1.) Element Selector - It selects by the name of the element. Ex. div{ css property here }.

2.) Class Selector - It selects by the name of the class given to the element. class selector used by "." symbol. Ex. .parent{ css property here }.

3.) Id Selector - It Selects by the unique(Id) name that given to the element. id selector used by "#" symbol. Ex. #id1{ css property here }.

4.) Group Selector - Using Group selector we can select more then one element or group of elements. Group selector used by "," Symbol. Ex. h1 , p , .child { css property here }.

5.) Universal Selector - Universal selector applied universally liked it's name it applies style to the whole code. Universal selector used by "*" Symbol. Ex. *{ css property here }.
